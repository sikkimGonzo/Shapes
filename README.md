# Shapes
Исходя из требований:
1. включены тесты
2. на основе самой простой реализации можно добавлять новые фигуры, наследующие IShape 
(кроме того есть, например, ITriangle, уже для более сложной иерархии с разными видами треугольников)
3. за счет интерфейсов площадь вычисляется в compile-time
4. включена проверка, является треугольник прямоугольным, для фигур, наследующих ITriangle.
